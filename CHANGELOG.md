## [v0.1.0] - 2026-07-05

### Highlights

- Batteries-included starter template for building AI agents with OmniAgent
- Pre-configured with 18 markdown skills, GitHub API skill, web search, and Meeting Facilitator role

### Added

- Starter bundle with skills and roles for quick agent setup ([`3f42906`](https://github.com/plexusone/omniagent-starter/commit/3f42906))
- Runnable agent in `cmd/agent/main.go` demonstrating skills, roles, and session storage ([`9d57798`](https://github.com/plexusone/omniagent-starter/commit/9d57798))
- Integration with `omniskill-pack` providing 18 markdown skills (git, docker, tmux, weather, etc.) ([`3f42906`](https://github.com/plexusone/omniagent-starter/commit/3f42906))
- GitHub SDK skill via `omni-github` for issues, PRs, and code search ([`3f42906`](https://github.com/plexusone/omniagent-starter/commit/3f42906))
- Web search skill via `omniserp` for Google/news search ([`9d57798`](https://github.com/plexusone/omniagent-starter/commit/9d57798))
- Meeting Facilitator role via `omnirole-facilitator` with action tracking ([`3f42906`](https://github.com/plexusone/omniagent-starter/commit/3f42906))
- SQLite session storage for persistent conversations ([`9d57798`](https://github.com/plexusone/omniagent-starter/commit/9d57798))
- Environment variable configuration for LLM providers, channels, and skills ([`9d57798`](https://github.com/plexusone/omniagent-starter/commit/9d57798))
