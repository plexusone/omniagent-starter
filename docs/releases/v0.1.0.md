# Release v0.1.0

**Release Date:** July 5, 2026

## Overview

Initial release of OmniAgent Starter - a batteries-included starter template for building AI agents with [OmniAgent](https://github.com/plexusone/omniagent).

This template provides a fully-configured agent out of the box, pre-configured with commonly used skills and ready to connect to messaging channels like WhatsApp, Telegram, and Discord.

## Highlights

- **Batteries-included starter template** for building AI agents with OmniAgent
- **Pre-configured with 18 markdown skills**, GitHub API skill, web search, and Meeting Facilitator role
- **Runnable agent** in `cmd/agent/main.go` demonstrating real-world usage patterns

## What's Included

### Skills

| Skill | Type | Description |
|-------|------|-------------|
| [omniskill-pack](https://github.com/plexusone/omniskill-pack) | Markdown | 18 skills: Git, Docker, tmux, Homebrew, weather, and more |
| [omni-github](https://github.com/plexusone/omni-github) | Compiled | GitHub SDK for issues, PRs, and code search |
| [omniserp](https://github.com/plexusone/omniserp) | Compiled | Web and news search via Serper/SerpAPI |

### Roles

| Role | Description |
|------|-------------|
| [omnirole-facilitator](https://github.com/plexusone/omnirole-facilitator) | Meeting facilitation with note-taking and action tracking |

### Infrastructure

- SQLite session storage for persistent conversations
- Environment variable configuration for LLM providers
- GitHub Actions shared workflows for CI/CD

## Quick Start

```bash
# Clone the repo
git clone https://github.com/plexusone/omniagent-starter.git
cd omniagent-starter

# Set your API key
export OPENAI_API_KEY="sk-..."     # Or ANTHROPIC_API_KEY

# Enable WhatsApp
export WHATSAPP_ENABLED=true

# Run the agent
go run ./cmd/agent gateway run
```

## Environment Variables

| Variable | Description |
|----------|-------------|
| `OPENAI_API_KEY` | OpenAI API key for GPT models |
| `ANTHROPIC_API_KEY` | Anthropic API key for Claude models |
| `GEMINI_API_KEY` | Google API key for Gemini models |
| `WHATSAPP_ENABLED` | Set to `true` to enable WhatsApp channel |
| `TELEGRAM_BOT_TOKEN` | Telegram bot token |
| `DISCORD_BOT_TOKEN` | Discord bot token |
| `GITHUB_TOKEN` | GitHub token for GitHub skill |
| `SERPER_API_KEY` | Serper API key for web search |
| `STORAGE_PATH` | SQLite path (default: `omniagent.db`) |
| `ENABLE_FACILITATOR_ROLE` | Set to `true` for Meeting Facilitator role |

## Programmatic Usage

Import the starter bundle into your own agent:

```go
import (
    "os"

    "github.com/plexusone/omniagent/agent"
    starter "github.com/plexusone/omniagent-starter"
)

func main() {
    bundle := starter.Default(starter.Config{
        GitHubToken:               os.Getenv("GITHUB_TOKEN"),
        FacilitatorActionTracking: true,
    })

    a, _ := agent.New(config,
        agent.WithSkillPack(bundle.SkillPack().FS()),
        agent.WithCompiledSkill(bundle.GitHubSkill()),
        agent.WithRole(bundle.FacilitatorRole()),
    )
}
```

## Requirements

- Go 1.26 or later
- An LLM API key (OpenAI, Anthropic, or Google)
